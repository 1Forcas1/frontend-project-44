# Description

This repository contains 5 brain games

## Minimum Requirements

Make sure you have Node.js version 13 or higher installed.

## Setup

```bash
make install 
```
## Installing the package on the system

```bash
npm link
```

## Game launch

* brain-even game

```bash
brain-even
```


* brain-calc game

```bash
brain-calc
```
* brain-gcd game

```bash
brain-gcd
```

* brain-prime game

```bash
brain-prime
```

* brain-progression game

```bash
brain-progression
```
